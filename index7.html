Aula de Eletrônica: Circuito com Botão e LED

Introdução:

Nesta aula, vamos aprender a montar um circuito simples utilizando um botão, um resistor, um LED, um Arduino e uma protoboard. O objetivo é fazer com que o LED acenda quando o botão for pressionado.


Teoria:



Tinkercad:

Código:
// C++ code
//
void setup()
{
pinMode(6, OUTPUT);
}

void loop()
{
digitalWrite(6, HIGH);
delay(10); // Delay a little bit to improve simulation performance
}



Conclusão:

Nesta aula, aprendemos a montar um circuito simples com um botão e um LED, controlado por um Arduino. Você pode experimentar modificando os valores dos componentes e o código para explorar diferentes comportamentos.
