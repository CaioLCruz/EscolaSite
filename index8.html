Circuitos Digitais e Introdução a C++/C#

Introdução:

Circuitos Digitais

Nesta aula, vamos aprender sobre os conceitos básicos de circuitos digitais, incluindo:

  • Entrada, processamento e saída
  • Portas lógicas (AND, OR, NOT)
  • Tabelas verdade
  • Circuitos elétricos com chaves, fontes, resistores e LEDs
  • Portas lógicas com ICs (7408, 7432, 7404)
  • Equações e diagramas das portas lógicas

Teoria:



:



Também vamos explorar a linguagem de programação C++/C# e criar um circuito simples com LED, resistor, protoboard e Arduino. O código será dividido em três partes:

  1. Declaração de variáveis e constantes
  2. Configuração (entradas, saídas, lógica e matemática)
  3. Repetições

Código de exemplo

 /* ----------------------------- Professor: Caio Da Cruz - Itinerário de Robótica - Curso de Arduino - Edusesc Taguatinga Norte - ----------------------------- */ // Declaração de Variáveis e Constantes
               const int LED = 13;
               // Configurações
               void setup(){
               pinMode(LED, OUTPUT);
               }
               // Repetições
               void loop(){
               digitalWrite (LED, HIGH);
               delay(1000); // 1 segundo
               digitalWrite (LED, LOW);
               delay(1000); // 1 segundo
               }

Neste código, estamos usando o pino 13 do Arduino para controlar um LED. No método setup(), configuramos o pino 13 como saída. No método loop(), alternamos o estado do LED entre ligado (HIGH) e desligado (LOW) a cada segundo, usando a função delay() para controlar o tempo.

Esperamos que esta aula tenha sido útil para você entender os conceitos básicos de circuitos digitais e a linguagem C++/C#. Fique à vontade para praticar e explorar mais exemplos!


Conclusão:

Nesta aula, aprendemos a montar um circuito simples com um botão e um LED, controlado por um Arduino. Você pode experimentar modificando os valores dos componentes e o código para explorar diferentes comportamentos.
