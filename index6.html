Semáforo (Arduíno)

Introdução

O objetivo desse projeto é criar um semáforo que pode ser controlado por um Arduino.

Componentes:

  • Arduino Uno
  • LEDs vermelhos, amarelos e verdes
  • Resistores
  • Conectores (jumpers)


Teoria:


Tinkercad:

Código:
// C++ code
int counter; void setup() { pinMode(10, OUTPUT); pinMode(1, OUTPUT); pinMode(3, OUTPUT); pinMode(8, OUTPUT); pinMode(9, OUTPUT); pinMode(2, OUTPUT); for (counter = 0; counter < 10; ++counter) { digitalWrite(10, LOW); digitalWrite(1, LOW); digitalWrite(3, HIGH); delay(1000); // Wait for 1000 millisecond(s) digitalWrite(8, HIGH); delay(3000); // Wait for 3000 millisecond(s) digitalWrite(8, LOW); digitalWrite(10, LOW); digitalWrite(9, HIGH); delay(2000); // Wait for 2000 millisecond(s) digitalWrite(9, LOW); digitalWrite(10, HIGH); digitalWrite(3, LOW); delay(1000); // Wait for 1000 millisecond(s) digitalWrite(1, HIGH); delay(3000); // Wait for 3000 millisecond(s) digitalWrite(1, LOW); digitalWrite(2, HIGH); delay(2000); // Wait for 2000 millisecond(s) digitalWrite(2, LOW); digitalWrite(3, HIGH); } } void loop() { delay(10); // Delay a little bit to improve simulation performance }



Conclusão:

Nessa aula, nós aprendemos a criar um semáforo com Arduino. O semáforo é um sistema de sinalização que ajuda a controlar o trânsito em uma cidade. Nesse projeto, nós usamos um Arduino Uno para controlar os LEDs vermelhos, amarelos e verdes que formam o semáforo.
Esse projeto demonstra como o Arduino pode ser usado para criar um sistema de sinalização mais complexo. Além disso, esse projeto pode ser ampliado para incluir mais funcionalidades, como a capacidade de controlar o tempo de duração dos LEDs. Esperamos que essa aula tenha ajudado a entender como criar um semáforo com Arduino.
